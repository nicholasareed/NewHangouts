Account Types

These are the major account types

  1. Banned (0): User cannot sign in at all
  2. Disabled (1): User can sign in, but not use apps
  3. Standard (2):User can sign in, and use all standard apps. This is given by default when you register.
  4. Moderator (3): User can sign in, and use all standard apps, plus a few extra permissions
  5. Admin (4): User has most control over the entire site. This is only given to the site creators and maybe few others

Administrators can promote or demote other users.
